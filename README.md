# 한복 타로 점 웹사이트

전통 한복 테마로 재해석한 타로 카드 점보기 웹사이트입니다.

## 기능

- 🔀 **카드 셔플**: 78장의 카드 중 랜덤으로 선택
- 🃏 **카드 표시**: 1장, 3장, 5장 중 선택하여 카드 뽑기
- 🔄 **정/역방향**: 각 카드가 랜덤으로 정방향 또는 역방향으로 표시
- 📖 **카드 해석**: 각 카드의 정/역방향 의미 자동 표시
- 🔁 **다시 뽑기**: 언제든지 새로운 카드를 뽑을 수 있음
- 📱 **반응형 디자인**: 웹과 모바일 모두에서 완벽하게 동작

## 카드 구성

### 메이저 아르카나 (22장)
- The Fool부터 The World까지 22장의 메이저 아르카나 카드

### 마이너 아르카나 (56장)
- **붓 (Wands)**: 의지, 창조, 영혼
- **다기 (Cups)**: 감정, 직관, 관계
- **부채 (Swords)**: 지성, 결정, 고통
- **옥/노리개 (Pentacles)**: 현실, 재물, 육체, 가문

각 수트마다 Ace(1)부터 10까지, 그리고 동자/동녀(Page), 선비(Knight), 궁주(Queen), 대감(King)까지 총 14장씩 구성되어 있습니다.

## 사용 방법

1. **카드 수 선택**: 1장, 3장, 5장 중 원하는 카드 수를 선택합니다.
   - **1장**: 원 카드 리딩 - 빠른 조언, 데일리 메시지
   - **3장**: 3카드 스프레드 - 과거 / 현재 / 미래
   - **5장**: 상황 분석 - 과거 / 현재 / 미래 / 조언 / 결과

2. **카드 뽑기**: "카드 뽑기" 버튼을 클릭합니다.

3. **카드 확인**: 카드가 자동으로 뒤집히며 정/역방향이 표시됩니다.

4. **해석 보기**: 모든 카드가 뒤집히면 자동으로 해석이 표시됩니다.

5. **다시 뽑기**: "다시 뽑기" 버튼을 클릭하여 새로운 카드를 뽑을 수 있습니다.

## 실행 방법

### 로컬 서버 실행

Python이 설치되어 있는 경우:
```bash
python -m http.server 8000
```

또는 Node.js가 설치되어 있는 경우:
```bash
npx http-server
```

그 후 브라우저에서 `http://localhost:8000`으로 접속하세요.

### 직접 열기

`index.html` 파일을 브라우저에서 직접 열어도 동작합니다. (단, 일부 브라우저에서는 CORS 정책으로 인해 JSON 파일 로드가 실패할 수 있습니다.)
